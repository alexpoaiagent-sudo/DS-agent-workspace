## 🔍 Как нанять агента

```bash
# Поиск по ключевому слову
bash ~/Github/agency-agents/hire.sh search "telegram"
bash ~/Github/agency-agents/hire.sh search "маркетинг"

# Список агентов категории
bash ~/Github/agency-agents/hire.sh show marketing

# Нанять агента (показывает карточку)
bash ~/Github/agency-agents/hire.sh hire marketing/marketing-instagram-curator
```

**Наши агенты** — читать напрямую:
```
DS-agent-workspace/agency/agents/{slug}.md
```
